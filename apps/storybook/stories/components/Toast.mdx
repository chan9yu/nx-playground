# Toast

## Usage

1. Toast를 노출할 영역에 `ToastProvider`를 추가합니다.

<Source
	dark
	language="jsx"
	code={`
<ToastProvider>
  <Router /> // e.g. React Router Dom
</ToastProvider>
  `}
/>

2. `useToast`를 사용하여 `Toast`를 노출합니다.

<Source
	dark
	language="jsx"
	code={`
const SomeComponent = () => {
  const { toast } = useToast();
  
  const handleClick = () => {
    toast({
      payload: {
        message: "toast!",
      },
    });
  };

return (<Button onClick={handleClick}>Toast Button</Button>);
};
`}
/>
